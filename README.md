# Family Store — Grocery Pickup System

Local grocery ordering platform. Customers browse products, place orders, and schedule pickup times. Store staff picks items and manages order fulfillment.

### Tech

- Next.js (App Router) + TypeScript
- Tailwind CSS
- Zustand (state management)

### Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
